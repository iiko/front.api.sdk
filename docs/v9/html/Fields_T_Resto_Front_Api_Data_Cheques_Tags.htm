 NameDescription
Barcode Format embedded text as barcode.
Br Line break. Any number of consecutive <br/>s tags will result in a single line break
Cell Single cell of the table with arbitrary content including other tags.
Cells Table cells, serialized line by line (LTR).
Center Align a content by the center.
Column Declares single column inside Columns collection.
Columns Declares a set of columns (Column).
CutText Excess text will be trimmed out.
Doc Root of the document.
Fill Overrides the character used to fill in the whitespace. A new character (or character set) is specified by the Symbols attribute.
Html HTML Formatting: Specifies HTML content for rendering within the CDATA section. This tag does not have native support and requires an external plugin for converting the 'html' into an 'image' tag.
Image Prints a monochromatic image represented as a base64 string. Supported attributes:
  • Align
  • ResizeMode
Justify Justify a content.
LargeFont Sets the largest possible font size of the content.
Left Align a content to the left.
Line Prints a line filled by specified Symbols attribute. Default symbol is "-".
LineCell Cell will be stretched into all columns of a parent table filled by specified Symbols attribute. Default symbol is "-".
Logo Prints image from non-volatile memory (NV image store) that was previously loaded. Using a numeric index.
MediumFont Sets medium font size of the content.
NewParagraph New paragraph. After applying it, the text following it will start on a new line, and if the preceding paragraph was formatted with width, then the last line of that paragraph will be aligned to the left.
NoBr Non-breaking space. Space character that prevents an automatic line break at its position.
Pagecut Page cut.
Pair Prints two values, first part is assigned to Left attribute and will be aligned to the left, second one is assigned to Right attribute and will be aligned to the right.
Pulse Outputs pulse to Drawer kick-out port.
QRCode Formatting embedded text as QR code.
Right Align a content to the right.
SmallFont Sets the smallest possible font size of the content, used by default.
Spacecolumn Empty column
SplitText Split text into lines on whitespaces.
Table Prints the table using two inner tags — Columns and Cells, first declares the layout of the table, latter provides cell values.
TextCell Single cell of the table with text content.
WrapText Split text into lines on any chars.
