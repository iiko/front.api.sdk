Remarks
Used for cancellation of operation when the reason of cancellation is obvious.
For example, user pressed "Cancel" button during collecting of required data - it's clear that operation will not be resumed.
No message box will be shown.
If the reason is not obvious, use PaymentActionFailedException that will show message box for user.
