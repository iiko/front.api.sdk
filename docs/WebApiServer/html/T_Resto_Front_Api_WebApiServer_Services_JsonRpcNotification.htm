JsonRpcNotification Class

Контракт JSON-RPC нотификации. JsonRequest не подходит для некоторых клиентов, например, для open-rpc, которые трактуют нотификации как id === undefined, но не id === null. А в контракте JsonRequest свойство Id не помечено как необязательное, поэтому включается в JSON со значением null.
